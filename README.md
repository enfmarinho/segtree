# Compilando e testando
Para compilar é necessário ter o gtest framework e cmake instalados.
````bash
cmake -S . -B build
cmake --build build
./build/segment_tree_test
``````
Os dois primeiros comandos irão compilar o código e o último irá rodar os testes.
